# Lab 1 — Tool Atlas & Accounts

Your first lab. It builds the **map** of the AI-SDLC landscape and gets your **accounts** ready — no app code yet (that starts Session 2). Full instructions are in the Session 1 lab guide your trainer shared; this folder gives you the starter files to fill in.

## What to produce in this folder

| File | What goes in it |
|---|---|
| `tool-atlas.md` | The phase × ecosystem grid from **Demo 1** — 9 SDLC phases × 4 ecosystems, in your own words, every cell filled, plus a one-line "what the AI does" note per phase. |
| `classification.md` | The 6 tools from **Demo 2**, each classified in one line on **two** axes: *generates or executes?* and *AI-powered or rule-based?* |
| `quiz.md` | Your short answers to **Self-quiz 1** (5 questions). |
| `accounts/` | Screenshots showing you're signed in to **at least** GitHub and Claude.ai (more tools is better — the set grows over the course). |

The three `.md` files are pre-seeded with blank skeletons — open each, read the note at the top, and fill it in. **Fill them yourself rather than copy-pasting a finished answer** — that's how the map sticks.

## Submit

```bash
git add labs/week-01
git commit -m "Lab 1: Tool Atlas, account screenshots, tool classification"
git push origin main
```

Then share **either** a link to your public fork's `labs/week-01/` folder, **or** open a Pull Request back to the course repo and share its URL. Your trainer will confirm the channel.
