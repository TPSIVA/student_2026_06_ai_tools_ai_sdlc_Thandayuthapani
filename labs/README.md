# Labs

Your weekly lab work lives here, one folder per week: `week-01/`, `week-02/`, … `week-16/`.

Each folder ships with starter files and a `README.md` telling you what to produce and how to submit. Do the work in **your own public fork**, commit it, and submit the link (or a PR) each week. Weekly sign-offs + the Week-16 capstone earn your certificate.

See the repo [README](../README.md) for the fork/clone/submit workflow.
